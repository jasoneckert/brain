To install this project, run the following commands in your Terminal app (/Applications/Utilities/Terminal):

sudo cp -f brain.bin /usr/local/bin
sudo chgrp staff /usr/local/bin/brain.bin 
sudo chmod 510 /usr/local/bin/brain.bin

This will copy the brain.bin program to your /usr/local/bin directory, and ensure that you and the staff group can run it.
